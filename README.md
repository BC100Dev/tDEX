# tDEX
tDEX itself is a native continuation of our old, beloved hacker application that makes
anyone feel like they're hacking the Pentagon,
[eDEX-UI](https://github.com/GitSquared/edex-ui). The reason that I decided to tackle this
entire rewrite is because I would crank up my Qt6 skills. And if you're wondering, no I did
not learn my lesson regarding "reinventing the damn wheel", I just like wasting my own damn
time. But hey, I will still include the customization that eDEX-UI also delivered, which is
through JSON files and such. Compatibility also matters to me. Also, this rewrite will also
have native APIs, if anyone likes to modify it even deeper because customization must go
brrrrrrr~
